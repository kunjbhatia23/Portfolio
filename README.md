# Kunj Bhatia - Personal Portfolio

![Portfolio Screenshot](https://placehold.co/800x400/0f172a/0ea5e9?text=Portfolio+Showcase)

This repository contains the code for my personal portfolio, a responsive single-page application built with modern web technologies. It's designed to showcase my skills and projects through an interactive UI, featuring smooth scroll animations, a project modal system, and a dynamic typing effect in the hero section.

[View the live portfolio here](https://your-live-url.com)

## 🛠️ Tech Stack

* **HTML5**
* **Tailwind CSS**
* **JavaScript (ES6+)**
* **Devicon** & **Bootstrap Icons** for logos.

---

## 🔧 How to Use

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/kunjbhatia23/Portfolio
    ```
2.  **Navigate to the directory:**
    ```bash
    cd Portfolio
    ```
3.  **Open `index.html` in your browser.**
